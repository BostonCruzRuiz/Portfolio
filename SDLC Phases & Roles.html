SDLC Phases & Roles

Here's The Roles For SDLC

Iteration

a set amount of time reserved for development

Simple Example: Giving A Set Amount Of Time To Get The App/Game Ready

Release

the distribution of the final version or newest version of a software application

Simple Example: The Release of a game that has a new version

Inception

The Project idea is being conceived and the initial planning is done.

Simple Example: Just Planning for how the project idea and how it works is going to get done
