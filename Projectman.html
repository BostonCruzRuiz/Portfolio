    The Word & Definition

    Unit Test

    The Verification Of A Small Piece Code Functions To Work Correctly

    Plan

    Decide On And Arrange In Advance

    Agile

    Being Able To Respond To Change In An Extremely Quick Manner

    Milestone

    A Certain Point Of A Projects Life That Is Used To Measure The Progress Of The Project And To Show How Close You Are To Your Ultimate Goal.

    Backlog

    An accumulation of tasks unperformed or materials not processed

    Task List in Github

    A task list a set of tasks that each render on a separate line with a clickable checkbox

    Sprint

    Think of is as a focused, mini-project aimed to achieve clear goals within a set time frame. A sprint is used for 1-4 weeks.

    Gantt Chart

    A Gantt chart in which a series of horizontal lines shows the amount of work done or production completed in certain periods of time in relation to the amount planned for those periods.

    Software Requirements

    Software requirements for a system are the description of what the system should do, the service or services that it provides and the constraints on its operation.

    Waterfall Methodology

    The Waterfall Methodology also known as the Waterfall model is a sequential development process that flows like a waterfall though all phases of a project (analysis, design, development, and testing, for example), with each phase completely wrapping up before the next phase begins.

    Maintenance Branch

    Maintenance Branch is a branch that is used to quickly patch production.

    Here's Some Old Words And Definitions From The Last Vocabulary Voc Week (That Was About Layers)

    Layer

    A Layer Communicates Parts Of The Software.

    Cross-Cutting Concerns

    Cross-Cutting Concerns Is When Programming An Application Is A Concern Where It Affects.

    Error - Handling

    The Processes Of Responding To And Recovering From Error Conditions In Your Program.

    Persistence Layer

    Is A Software System That Is Similar To A Database And Saves/Gets Data From A Storage.

    Error - Messaging

    An Error Message Is The Information Displayed When An Unforeseen Problem Occurs Usually On A Computer Or Some Other Devices.

    Logging

    Logging Is Act Of Keeping A Log Events That Occur In A Computer System.

    Authorization

    Allowing Access For Someone To Get Data.

    UI

    It Is The Computer Waiting For The User To Talk To The Computer

    Application Software

    A Program Designed To Handle Specific Tasks For Users

    Software

    Software Is A Set Of Instructions Data Of Programs Used To Operate Computers And Execute Specific Tasks For Users

    Logic Layer

    The Part Of Program When Classes Determine The Information To Solve Problems Manipulate That Information And Return Results To The Presentation Layer For Formatting.

    Authentication

    Verifying The Identity Of A User Process Or Device Often As A Prerequisite Allowing Access To Resources In An Information System.
