Smoke Testing (In Software Testing)

It’s A Type Of Method That’s Used In Software Testing To See If A Build Is Stable Or Not It’s Also One Of 5 Other Tests That Can Be Used.

Mocks, Mocking

A Test Double That Stands In For Real Implementation Code

Acceptance Testing

Software Testing That Evaluates Whether A System Meets It's Business And User Requirements

Test Driven Development

A Way Of Writing Code That Involves Writing An Automated Unit-Level Test Case

Load Testing

Identify System Lag, Page-Load Issues, And Anything Else That Might Go Awry When Multiple Users Access An Application

Regression Testing

A Software Testing Technique That Re-Runs Non-Functional And Functional Tests To Ensure That A Software Application Works As Intended

Integration Testing

Test How Parts Of The Application Work Together As A Whole

Unit Testing

Block Of Code That Verifies The Accuracy Of A Smaller, Isolated Block Of Application Code

Content Management System Also Called (CMS)

A Graphical User Interface With Tools To Create Websites

Non-Functional Testing

A Type Of Software Testing That Verifies Non-Functional Aspects Of The Product

Functional Testing

It Verifies That Each Function Of A Software App Works In Conformance With The Requirement And Specification

Web Framework

A Software Framework That Is Designed To Support The Development Of Web Applications
