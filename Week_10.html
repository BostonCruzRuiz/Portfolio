css specificity order ( important id inline class)

Inline styles will always have the highest specificity level. Second in this order are IDs, like #content . So any selector using an ID will have the second highest specificity level. Classes, pseudo-classes, and attributes are third in this order.

css border-size: top right bottom left

When one value is specified, it applies the same width to all four sides. When two values are specified, the first width applies to the top and bottom, the second to the left and right.
